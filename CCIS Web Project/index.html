Courses Offered

The College of Computing and Information Sciences
offers varieties of courses

BSIT

BSIT is a 4-year program which focuses on the study of computer utilization and computer software to plan, install, costumize, operate, manage, administer, and maintain information technology infrastucture

BSCS

BSIT is a 4-year program that includes the study of computing concepts and theories algorithmic foundations and new developments in computing. Graduates should be able to design algorithmically complex software new and effiecient algorithms for solving computer problems.

BSIS

BSIS is a 4-year program that prepares students to be able to help the organization on how information and technology-enabled business processes can be used as strategic tool for achieving competitive advantages.
